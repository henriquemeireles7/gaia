# gstack — vendored skill foundation

Per vision §Harness, Gaia ships with three foundation skills vendored from gstack:

- `plan/SKILL.md` — implementation strategy planning.
- `review/SKILL.md` — PR diff review.
- `qa/SKILL.md` — web-app QA testing.

These are vendored (not installed) so the template runs on day one without an external dependency on gstack itself. Users with `gstack` installed globally already have these as user-level skills; the vendored copies guarantee the template stands alone.

The current files in this folder are placeholders. Fully-vendored copies are tracked under the `gstack-vendor` initiative.
